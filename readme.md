# Adonis Frontend Orderin application

## Setup

Use the adonis command to install the blueprint

```bash
git clone https://github.com/syaifulislam/orderin-app-frontend.git
```

change `.env.example` to `.env`.

```bash
git clone https://github.com/syaifulislam/orderin-app-frontend.git
```

### Start

```bash
php artisan serve
```
